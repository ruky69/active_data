Friendly UI in Sheets

Launch from Extensions → ActiveData, configure once, and reuse your sheet.

How it works

1Install & launch
Install from Marketplace, then open Sheets → Extensions → ActiveData.
2Connect & choose fields
Paste your ActiveCampaign API URL & key, pick campaigns and columns.
3Sync or schedule
Run a sync now, or set an hourly/daily/weekly schedule.
